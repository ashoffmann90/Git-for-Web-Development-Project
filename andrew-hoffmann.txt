1. What is the difference between an inline element and a block element?
Inline elements do not start on a new line, and will only take up as much width as necessary.
Block elements will always start on a new line, and will take up the full width of the element.

2. What happens when an element is positioned absolutely? 
It is positioned in relation to the nearest ancestor, instead of in relation to the viewport (like fixed positioning).
If there are no ancestors, it uses the viewpoint.
